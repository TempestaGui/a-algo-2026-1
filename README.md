# Análise de Algoritmos

Durante o desenvolvimento das atividades, estou implementando as soluções **tanto em Java quanto em Python**.

Sei que a atividade solicitada deve ser realizada em **Python**, por isso criei uma pasta separada chamada **`PYTHON`**, onde estarão as implementações utilizadas para a avaliação.

Além disso, também estou implementando os mesmos algoritmos em **Java**, com o objetivo de **praticar e aprofundar meus estudos na linguagem**.

## Estrutura do Projeto
```
dever_N (onde N representa o numero da atividade)
│
├── PYTHON
│ └── Implementações em Python (utilizadas para avaliação) 
│
└── JAVA
└── Implementações em Java (utilizadas para estudo)
```


## Objetivo

Comparar o desempenho de algoritmos de ordenação utilizando diferentes linguagens de programação e compreender melhor o comportamento e a eficiência desses algoritmos.
